# v0.2.1

## 播放

- 修复播放进度条与时间不随播放推进的问题。
- 冷启动可恢复上次曲目与进度（暂停展示，不自动出声）。
- 响度均衡增加 +6 dB 预增益，缓解部分曲目音量偏小。
- 沉浸式播放页不再展示「正在补充歌曲信息」类提示，避免打扰听歌。

## 性能

- 歌曲、队列、歌单等长列表改为虚拟化渲染，降低滚动卡顿。
- 扫描完成后批量持久化曲库，减少写入开销。
- 队列按 songId 索引解析，避免重复线性查找。
- 优化 AMLL 歌词索引匹配与运行时缓存边界；隐藏播放器时降低渲染开销。

## 界面与体验

- 建立「暗场听席」设计系统与 HeroUI primary 主色。
- 接入 npm `happier-ui` 语义组件与 `--h-*` token；业务图标统一为 Lucide + `HIcon`。
- 歌词页浮动翻译/播放按钮改为交互后显示、空闲自动隐藏。
- 统一导航与播放相关图标语义（队列、顺序/随机、专辑/艺人/音源等）。
