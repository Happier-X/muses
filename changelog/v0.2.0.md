# v0.2.0

## 播放

- 新增 **音量均衡**（ReplayGain 轻量）：扫描/懒读标签时解析 track ReplayGain，播放时按 dB 换算并限制在插件 volume 范围（约 0.1–1.0）；无标签不调整。
- 设置页增加「音量均衡」开关（默认开启），可即时作用于当前曲。

## 图标与 UI

- 底部播放条与沉浸式主控（上一曲 / 播放暂停 / 下一曲）使用 **fill 实心** 图标；列表「播放全部」等仍为线框。
- 统一「打开队列」「歌单」相关入口图标为 `list`（ListMusic）；顺序播放模式仍用 `listOutline`，避免同语义混用。

## 歌曲列表

- 「跳转到当前播放」时通过 `scroll-margin-top` 避开粘性顶栏（标题 + 随机播放全部），当前曲主信息不再被挡住。
