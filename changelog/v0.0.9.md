# v0.0.9 (2026-07-12)

## 新增

- 进度条展示已缓冲区域（类似视频播放器）。
- WebDAV 渐进下载：首包可播即开始播放，缓冲条随下载增长。
- 拖动进度与歌词点击仅允许在已缓冲区间内跳转，避免 seek 到未缓冲区。

## 修复

- seek 到未缓冲进度时不再误触发自动下一曲（seek 保护窗 + 自然结尾判定）。
- 本地就绪曲目缓冲视为满条，可全长 seek。
- 同步 Android versionName / versionCode 与 package-lock 版本号。

## 说明

v0.0.8 tag 因 Android 版本号未同步（仍为 0.0.6）而跳过，本版本等价于 v0.0.8 内容并修复版本号。
