# v0.1.2 (2026-07-13)

## 新增

- 播放时在线匹配封面：iTunes → 酷我 → QQ → 网易云 → 酷狗 → 咪咕，成功后缓存为本地 URI 并写回曲库。
- 播放时在线补缺歌手、专辑；弱标题（与文件名相同）可在相关命中时写回歌名。
- 多源在线歌词回退链：amll TTML → 酷我/QQ/网易/酷狗/咪咕 → LRCLIB → 本地 LRC。
- 网易云优先 YRC、QQ 优先 QRC 逐字歌词（AMLL 解析）；无逐字时降级行级 LRC。
- 统一 `matchOnlineLyrics` 编排；在线歌词不写回 `muses:songs`。

## 优化

- 封面与文本元信息仅在本地缺失/不足时补全；封面禁止 `data:`/base64 进库。
- HTTP 4xx/5xx 不再错误回退二次 fetch，减少双请求与测试实网污染。
- LRCLIB 仅使用带时间轴的 `syncedLyrics`，并带合规 User-Agent。

## 说明

- 酷狗 KRC / 咪咕 MRC / 酷我 lyricx 暂不做平台解密，这些源以行级 LRC 为主。
- 网易云并非每首歌都有 YRC；有则优先逐字。
