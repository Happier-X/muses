# v0.1.0 (2026-07-12)

## 新增

- 接入 amll-ttml-db：开播或切歌后自动匹配在线 TTML 逐词歌词。
- 在线 TTML 歌词优先展示；匹配失败时自动回退内嵌歌词或同目录 LRC。
- 歌曲页顶部增加「随机播放全部」按钮。
- 歌曲页右下角增加「跳转到当前播放歌曲」悬浮按钮。

## 优化

- 在线歌词支持歌名、歌手和专辑综合匹配，并加入运行时缓存与快速切歌防串词保护。
- 歌词页增加在线歌词匹配中、未匹配到歌词等状态提示。
- WebDAV 音源扫描默认关闭「读取音乐标签」，本地音源仍默认开启。
- 平板歌曲列表改为单列展示。

## 修复

- 修复平板 MiniPlayer 悬空，宽屏布局下播放条贴底显示。
- 修复平板沉浸式播放页封面在矮屏比例下被拉伸的问题。
- 修复歌词区域上下滑动与沉浸式播放页下滑关闭手势冲突的问题。
