# 调整底部播放条样式

## 背景

当前底部播放条是浮动卡片样式，只在有歌曲或错误时显示。用户希望它改成固定底栏样式，并且始终展示。

## 目标

- 底部播放条始终展示在非播放器/队列页。
- 播放条占满宽度，没有圆角。
- 左侧展示封面。
- 中间展示歌曲名和 `艺术家 - 专辑`。
- 右侧展示播放/暂停按钮和播放队列按钮。
- 保持点击播放条进入播放器页的能力。
- 保持播放/暂停、停止/队列等既有播放能力不回归。

## 范围

包含：

- 修改 `src/components/MiniPlayer.vue` 的模板、样式和按钮布局。
- 如当前没有歌曲，展示稳定占位信息和占位封面。
- 队列按钮跳转到 `/queue`。
- 构建验证。

不包含：

- 重构播放器状态管理。
- 新增完整队列编辑功能。
- 发布新版本。

## 验收标准

- [ ] 底部播放条在非播放器/队列页始终可见。
- [ ] 播放条横向占满屏幕宽度且没有圆角。
- [ ] 左侧展示封面或占位封面。
- [ ] 中间展示歌曲名与 `艺术家 - 专辑`。
- [ ] 右侧展示播放/暂停按钮和播放队列按钮。
- [ ] 点击播放条主体进入播放器页。
- [ ] 点击播放/暂停不触发进入播放器页。
- [ ] 点击队列按钮进入队列页且不触发进入播放器页。
- [ ] `npm run build` 通过。
