# 修复手机端底部导航栏消失

## 背景

发布 `v0.0.3` 后，用户发现手机端底部导航栏不见了。该问题发生在前一次 Android/MuMu 导航布局修复之后，疑似由 `TabsPage.vue` 响应式布局调整引入。

## 目标

- 恢复手机/窄屏下底部 tab 导航栏显示。
- 保持平板宽度下左侧导航栏布局正常。
- 避免重新引入 `ion-split-pane` / `ion-menu` 导致的 Android WebView 白屏问题。

## 范围

包含：

- 排查并修复 `src/views/TabsPage.vue` 中移动端 tab bar 显示问题。
- 构建验证。
- 如需要，安装到 MuMu 验证。

不包含：

- 重新设计导航结构。
- 发布新版本，除非用户后续明确要求。

## 验收标准

- [x] 窄屏/手机端底部 tab 可见并可点击。
- [x] 平板宽度下仍只显示一个左侧导航栏。
- [x] 右侧/主内容正常显示。
- [x] `npm run build` 通过。
