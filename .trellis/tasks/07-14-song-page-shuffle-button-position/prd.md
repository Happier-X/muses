# 歌曲页随机播放按钮移到列表与 navbar 之间

## 背景

歌曲页面已有“随机播放全部”按钮。目前按钮位置不符合期望，需要调整到歌曲列表与底部 navbar 之间，让操作入口贴近列表但不遮挡导航。

## 需求

- 调整歌曲页面“随机播放全部”按钮的位置。
- 按钮应位于歌曲列表和 navbar 之间。
- 保持现有随机播放逻辑、可访问性标签、禁用态和测试语义不变。
- 不引入额外业务逻辑变化。

## 验收标准

- 歌曲页中“随机播放全部”按钮显示在歌曲列表之后、navbar 之前。
- 点击“随机播放全部”的现有行为保持不变。
- 无歌曲时按钮仍保持禁用。
- 相关单元测试通过，或在现有测试语义需要更新时只更新位置相关断言。
