# 歌词页浮动按钮按需显示

## Goal

沉浸式歌词页左下翻译、右下播放/暂停两个浮动按钮默认隐藏；用户点击页面或滑动歌词时再显示，减少对歌词阅读的遮挡。

## Background

- 按钮位于 `PlayerPage.vue` 的 `.lyric-floating-actions`。
- 窄屏：翻译 + 播放/暂停；平板（≥768px）：仅翻译（播放键已隐藏）。
- 当前始终可见，会压住底部歌词。

## Requirements

- 默认隐藏两个浮动按钮（无交互静止态）；隐藏时不可点（`pointer-events: none`）。
- **仅歌词面板内**交互触发显示：
  - 点击歌词页（含空白/歌词区，在不破坏现有手势的前提下）
  - 滑动浏览歌词
- 显示后可操作：翻译开关、播放/暂停语义不变；操作浮动按钮重置空闲计时。
- **空闲 3 秒**无继续交互后再次自动隐藏。
- 显示/隐藏使用约 **150–200ms** 短 fade；隐藏态与过渡中不可点。
- **切回控制页**或**关闭播放器 overlay**时立即隐藏并清除 timer；再次进入歌词页从隐藏开始。
- 平板仅翻译键时同样适用按需显示。
- 不破坏：歌词行点击 seek、横向切面板、下滑关闭 overlay、歌词区不触发下滑关闭。

## Decisions

- 空闲自动隐藏延迟：**3 秒**。
- 隐藏态 **不接收点击**。
- **短 fade** 约 150–200ms。
- 仅歌词面板交互触发；离开歌词面板 / 关 overlay **立即隐藏并清 timer**。

## Acceptance Criteria

- [x] 打开歌词页后静止时浮动按钮不可见且不可点
- [x] 点击歌词页或滑动歌词后按钮淡入出现且可点
- [x] 空闲 3 秒后按钮淡出隐藏
- [x] 操作翻译/播放按钮会重置 3 秒计时
- [x] 切回控制页或关闭 overlay 后按钮为隐藏且 timer 已清
- [x] 翻译开/关、播放/暂停行为不回归
- [x] 歌词 seek / 横滑切页 / 下滑关闭不回归
- [x] 单测覆盖显示触发、空闲隐藏、离开面板强制隐藏
- [x] lint / 相关 unit / build 通过
